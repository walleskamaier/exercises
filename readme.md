## 📋 About

- Code Signal
- Exercism
- freeCodeCamp
- And more


## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/walleskamaier/exercises.git
   cd exercises
   ```

2. Run the exercise:
   ```bash
   node reverse-number-digits.js
   ```
